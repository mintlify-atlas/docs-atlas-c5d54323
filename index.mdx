DefinitelyTyped is the community repository powering TypeScript's `@types` ecosystem. With over 8,000 type definition packages maintained by contributors worldwide, it lets you add TypeScript type safety to nearly any JavaScript library with a single `npm install`.

## What is DefinitelyTyped?

TypeScript requires type information to provide autocomplete, error checking, and refactoring support. Many JavaScript libraries ship without TypeScript types. DefinitelyTyped solves this by hosting community-maintained `.d.ts` declaration files for those libraries.

When you install `@types/node` or `@types/react`, you're using DefinitelyTyped. The `master` branch is automatically published to the `@types` scope on npm by [DefinitelyTyped-tools](https://github.com/microsoft/DefinitelyTyped-tools).

## Get started in seconds

```bash
npm install --save-dev @types/node
```

For most packages, this is all you need. TypeScript will automatically pick up the types.
